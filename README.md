# Rere-Expense-App-2 (RereExpense)

RereExpense is an advanced mobile application built to track your daily, monthly, and yearly expenses seamlessly.

## Features
- Track incoming and outgoing funds.
- Categorize transactions for better insights.
- Intuitive graphs and summaries to understand spending habits.
- Secure, fast, and accessible offline.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npx expo start
   ```
